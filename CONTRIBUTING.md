# Contributing to CarbonWise

Thank you for your interest in contributing to CarbonWise.

CarbonWise is a sustainability-focused web application designed to help users understand, track, and reduce their carbon footprint through data-driven insights and actionable recommendations.

## Getting Started

### Prerequisites

* Node.js 20+
* npm
* Git

### Installation

Clone the repository:

```bash
git clone https://github.com/Sarang-Salunkhe/carbonwise-ai.git
cd carbonwise-ai
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

## Development Workflow

1. Create a new branch for your feature or fix.
2. Make your changes.
3. Test your changes locally.
4. Ensure all quality checks pass.
5. Submit a pull request.

## Quality Requirements

Before submitting changes, ensure the following commands complete successfully:

```bash
npm run lint
npm run test:run
npm run build
```

## Coding Standards

* Follow existing project structure and naming conventions.
* Keep components modular and reusable.
* Write clear and maintainable code.
* Add tests for new business logic where applicable.
* Preserve accessibility and responsive design standards.

## Reporting Issues

When reporting bugs, please include:

* Expected behavior
* Actual behavior
* Steps to reproduce
* Browser and device information
